Becoming a coder extraordinaire!

I'm coding this blog! Crazy times!

So far, it's actually pretty simple but there is a lot to remember, kind of like learning a new language. I feel like it is one of those things that you have to just practice every day and keep doing new projects in order to get better at it.

I have learnt SQL (Structured Query Language) at work and I remember the first time I saw it. I was thinking, there is no way am I going to be able to do that but actually I committed an hour or so a week to learning it, including playing around in the databases challenging myself to find new data. Soon enough I was all over it, and even after a few months became a "Go-to guy" for SQL. I feel like this is going to be the same. The only challenge is that I don't have to do this for work...
